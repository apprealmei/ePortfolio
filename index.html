MIPS assembly language "Rock, Paper, Scissors" Game

The MIPS assembly language "Rock, Paper, Scissors" game primarily serves an educational purpose, demonstrating basic concepts such as user input handling, random number generation, conditional branching, and control flow. The program allows the user to play a game of "Rock, Paper, Scissors" against the computer. The game is a "best of three" format, meaning the first to win three rounds is the overall winner. Additionally, if the user loses a round, they are given a punishment, which is randomly selected as either a "Truth" or "Dare."

Laravel (Lumbab_Project)

Laravel is a PHP web framework that is available as a free download that focuses on the development of contemporary web applications. It adheres to the model-view-controller (MVC) architectural pattern, and seeks to improve the development process and productivity of developers. Laravel has a number of features and components that are built-in such as routing, database migrations, authentication and an ORM (Eloquent) to work with the database. Its syntax is said to be elegant and simple to use with many supporters in the community. Laravel is just as often utilized for creating simple websites as well as large scale enterprise applications.
